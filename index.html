• You need node
  • You install Grunt on a per-project basis
  • Navigate to your project and create a file package.json
  • Add some setup code to this file: 
    
                {
                  "name": "example-project",
                  "version": "0.1.0",
                  "devDependencies": {
                    "grunt": "~0.4.1"
                  }
                }
  • Navigate to your project directory in a terminal, if you haven't already done so.
  • Install the dependencies in the local node_modules folder by running the command npm install
  • Magic will happen and a new folder will appear in your directory called node_modules
  • Now install Grunt in the project with the command npm install -g grunt-cli (you may need to do this with sudo)
  • Install modules: 
    npm install grunt-contrib-watch --save-dev
    npm install grunt-contrib-sass --save-dev
  • Create a Gruntfile. It's a javascript file named Gruntfile.js. Put the following into it: 
    
                module.exports = function(grunt) {

                    // 1. All configuration goes here 
                    grunt.initConfig({
                        pkg: grunt.file.readJSON('package.json'),

                        sass: {
                            dist: {
                                options: {
                                    style: 'compressed'
                                },
                                files: {
                                    'css/build/main.css': 'css/main.scss'
                                }
                            } 
                        },

                        watch: {
                            css: {
                                files: ['css/*.scss'],
                                tasks: ['sass'],
                                options: {
                                    spawn: false,
                                }
                            }
                        }

                    });

                    // 3. Where we tell Grunt we plan to use this plug-in.
                    grunt.loadNpmTasks('grunt-contrib-watch');
                    grunt.loadNpmTasks('grunt-contrib-sass');

                    // 4. Where we tell Grunt what to do when we type "grunt" into the terminal.
                    grunt.registerTask('default', ['watch', 'sass']);

                };
